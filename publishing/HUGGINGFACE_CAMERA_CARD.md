# MCP Camera Watch

Local-first MCP server for camera watch instructions, scene descriptions, and visual condition alerts.

Built by Steve Villari and Villocity Labs.

GitHub: https://github.com/Villocity-Labs/mcp-camera-watch

## Tool Ideas

- `camera_list`
- `camera_snapshot`
- `camera_describe`
- `camera_evaluate_once`
- `camera_watch_create`
- `camera_watch_start`
- `camera_watch_stop`
- `camera_watch_status`
- `camera_alerts_list`

This project is intended to pair with MCP Printer while remaining useful as a standalone camera monitoring MCP.
