# MCP Printer

Local-first MCP and OpenClaw tools for OctoPrint and Moonraker/Klipper 3D printers.

Built by Steve Villari and Villocity Labs.

GitHub: https://github.com/4kmg4gr769-sys/mcp-printer

## Features

- List configured printers
- Check printer status
- Upload G-code
- Start already-uploaded jobs
- Pause, resume, cancel
- Emergency stop
- OctoPrint support
- Moonraker/Klipper support
- Python stdio MCP server
- Native OpenClaw plugin wrapper

This project is local-first. Printer URLs and API keys should remain on the operator's machine.
